## 📖 About

Clarifi is a budget tracking application currently in development, built with a Spring Boot backend. The goal is to help users log expenses, categorize spending, and get a clear picture of their financial health — without the clutter of complex finance apps.

> 🚧 **This project is a work in progress.** Core backend structure is being built out; features listed below reflect the current roadmap.

## ✨ Planned Features

- 💵 Log and categorize daily expenses
- 📊 Visual breakdown of spending by category
- 📅 Monthly budget tracking and alerts when nearing limits
- 🔐 Secure user authentication (JWT-based, similar to prior projects)
- 🗄️ Persistent storage with a relational database (MySQL/MariaDB)
